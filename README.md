# Sandstorm-Simulator
A Guild Wars 2 Sandstorm simulator. 

A Sandstorm simulator using the drop rates from the Sandstorm Flush Hand as of 12/23/25 (https://wiki.guildwars2.com/wiki/Sandstorm_Flush_Hand/Drop_rate), combined with the live ectoplasm cost (provided via TP API at runtime) to estimate the profits (or losses) of playing as many hands of Sandstorm as your heart desires. 

(Y)es gives 1 flush hand at a time, (W)e Ball rolls 100 hands at once.
